## 7. ESP32-S3 接线

> 来源：`lgfx_conf.h` + STM32 例程参考

| ESP32-S3 GPIO | TFT 引脚 | 信号 | 说明 |
|:---:|------|------|------|
| **GPIO14** | Pin 3 (SCL) | SPI Clock | SPI3_HOST 经 GPIO 矩阵映射 |
| **GPIO13** | Pin 4 (SDA) | SPI MOSI | SPI3_HOST 经 GPIO 矩阵映射 |
| **GPIO12** | Pin 5 (RES) | Reset | 硬件复位，低有效 |
| **GPIO11** | Pin 6 (DC) | Data/Cmd | GPIO 单独控制 |
| **GPIO10** | Pin 7 (CS) | Chip Select | 片选 |
| **3.3V** | Pin 2 (VCC) | Power | 逻辑电源 |
| **3.3V** | Pin 8 (BLK) | Backlight | 背光常亮 |
| **GND** | Pin 1 (GND) | Ground | 地 |

```
ESP32-S3                      TFT-LCD (8-Pin)
┌──────────────┐              ┌────────────────┐
│          GND ├──────────────┤ GND (Pin 1)    │
│         3.3V ├──────────────┤ VCC (Pin 2)    │
│       GPIO14 ├──────────────┤ SCL (Pin 3)    │
│       GPIO13 ├──────────────┤ SDA (Pin 4)    │
│       GPIO12 ├──────────────┤ RES (Pin 5)    │
│       GPIO11 ├──────────────┤ DC  (Pin 6)    │
│       GPIO10 ├──────────────┤ CS  (Pin 7)    │
│         3.3V ├──────────────┤ BLK (Pin 8)    │
└──────────────┘              └────────────────┘
```

### GPIO 矩阵分析

ESP32-S3 的 GPIO10~14 同时也是 FSPI (SPI2) 默认引脚。当前接线**不直接兼容 FSPI**，但可通过 **SPI3_HOST + GPIO 交换矩阵** 正常工作：

| GPIO | FSPI 默认 | TFT 当前 | 兼容性 |
|------|-----------|----------|--------|
| GPIO10 | FSPICS0 (CS) | TFT CS | ✅ 天然匹配 |
| GPIO11 | FSPID (MOSI) | TFT DC | ⚠️ 被 DC 占用，不能用于 FSPI |
| GPIO12 | FSPICLK | TFT RES | ⚠️ 被 RES 占用 |
| GPIO13 | FSPIQ (MISO) | TFT SDA | ✅ 矩阵重映射为 MOSI |
| GPIO14 | FSPIWP | TFT SCL | ✅ 矩阵重映射为 SCLK |

### LovyanGFX 配置

```cpp
cfg.spi_host = SPI3_HOST;           // 使用 SPI3，避开 FSPI 默认引脚冲突
cfg.pin_sclk = 14;                  // 矩阵映射 SCLK → GPIO14
cfg.pin_mosi = 13;                  // 矩阵映射 MOSI → GPIO13
cfg.pin_miso = -1;                  // 不使用 MISO
cfg.pin_dc   = 11;                  // DC 普通 GPIO
cfg.freq_write = 80000000;          // 80MHz SPI
cfg.dma_channel = SPI_DMA_CH_AUTO;  // 自动 DMA
```

### 注意事项

1. **供电电压**：3.3V，不可接 5V
2. **背光**：BLK 直连 3.3V 常亮，也可接 GPIO 通过 PWM 调光
3. **SPI 模式**：CPOL=0, CPHA=0（模式 0），MSB 优先
4. **配色**：`cfg.invert = true`（不反色会显示异常），`cfg.rgb_order = false`（BGR 顺序）

## 8. ST7789V2 初始化序列

> LovyanGFX `Panel_ST7789` 类内置初始化，已在此屏验证通过

LovyanGFX 自动发送的关键命令：
```
SLPOUT (0x11) → 延时 130ms     # 退出睡眠
COLOR_MODE (0x3A) → 0x55      # 16bit/pixel RGB565
GCTRL (0xB7) → 0x35            # 栅极控制
VCOMS (0xBB) → 0x28            # VCOM 设置
INVON (0x21)                   # 反色使能
DISPON (0x29)                  # 开显示
```

> 与 STM32 例程的初始化序列略有不同（VCOM=0x28 vs 0x32），已验证无影响。

## 11. DIJI-NES 渲染流水线参考

DIJI-NES 使用双 SPI 总线隔离（TFT=SPI3_HOST, SD=FSPI），本项目仅用 TFT 可只用 SPI3_HOST。渲染流程为：NES PPU 输出 → `LGFX_Sprite` 帧缓冲 → `pushSprite` 推屏，60FPS。
