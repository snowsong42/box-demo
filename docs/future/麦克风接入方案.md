# I2S 麦克风接入方案

> ✅ **已实现**（2026-06）：实际采用 **I2S1 独立端口** 而非本文档的 I2S0 全双工方案。详见 [录音子系统.md](../guides/录音子系统.md)。
>
> 以下为原始设计文档，保留供参考。
>
> I2S0 全双工：TX → MAX98357A 播放 + RX → MEMS 麦克风采集。硬件只需加一根 DIN 线。

---

## 1. 硬件资源确认

### 1.1 I2S 外设

| 资源           | 占用情况             | 能力                                            |
| -------------- | -------------------- | ----------------------------------------------- |
| **I2S0** | TX 已用（GPIO5/4/6） | ESP32-S3 支持**全双工**，TX+RX 可同时工作 |
| I2S1           | 空闲                 | 备选方案（如需独立时钟域）                      |

当前配置：

```cpp
i2s_new_channel(&chan_cfg, &tx_chan, NULL);  // 仅 TX
```

改为全双工：

```cpp
i2s_new_channel(&chan_cfg, &tx_chan, &rx_chan);  // TX + RX
```

## 2. 硬件接线

```
ESP32-S3                    I2S MEMS 麦克风 (INMP441)
┌──────────────┐           ┌──────────────────┐
│ GPIO5 (BCLK) ├─────┬─────┤ SCK              │  ← 与扬声器共享
│ GPIO4 (WS)   ├───┬─┼─────┤ WS               │  ← 与扬声器共享
│ GPIO6 (DOUT) ├─┐ │ │     │                  │
│ GPIO1 (DIN)  ├─┼─┼─┼─────┤ SD (DATA)        │  ← 新增
│ 3.3V         ├─┼─┼─┼─────┤ VDD              │
│ GND          ├─┼─┼─┼─────┤ GND + L/R        │  L/R 接地=左声道
└──────────────┘ │ │ │     └──────────────────┘
                 │ │ │
                 │ │ └─── MAX98357A BCLK (已有)
                 │ └───── MAX98357A LRCLK (已有)
                 └─────── MAX98357A DIN (已有)
```

> BCLK 和 LRCLK 是总线信号，一个 I2S 主机可以驱动多个从设备共享时钟线。

### 兼容的麦克风模块

| 型号              | 接口         | 位深   | 采样率   | 备注                  |
| ----------------- | ------------ | ------ | -------- | --------------------- |
| **INMP441** | I2S Standard | 24-bit | 8k~44.1k | 最常见、便宜（~5元）  |
| ICS-43434         | I2S Standard | 24-bit | 8k~96k   | 低噪声                |
| SPH0645LM4H       | I2S Standard | 24-bit | 32k~64k  | 注意：需特殊 L/R 配置 |

## 3. 软件改动

### 3.1 I2S 初始化（`app_main`）

```cpp
// 改前：
i2s_new_channel(&chan_cfg, &tx_chan, NULL);

// 改后：
i2s_chan_handle_t rx_chan = nullptr;
i2s_new_channel(&chan_cfg, &tx_chan, &rx_chan);

i2s_std_config_t std_cfg = {
    .clk_cfg  = I2S_STD_CLK_DEFAULT_CONFIG(22050),
    .slot_cfg = I2S_STD_PHILIPS_SLOT_DEFAULT_CONFIG(
                    I2S_DATA_BIT_WIDTH_16BIT,
                    I2S_SLOT_MODE_MONO),
    .gpio_cfg = {
        .mclk  = I2S_GPIO_UNUSED,
        .bclk  = GPIO_NUM_5,
        .ws    = GPIO_NUM_4,
        .dout  = GPIO_NUM_6,
        .din   = GPIO_NUM_1,   // ← 新增
        .invert_flags = { .mclk_inv = 0, .bclk_inv = 0, .ws_inv = 0 },
    },
};

// TX 通道（已有）
i2s_channel_init_std_mode(tx_chan, &std_cfg);
// RX 通道（新增）
i2s_channel_init_std_mode(rx_chan, &std_cfg);
```

### 3.2 音频启用/停用

```cpp
// 进入子功能时（各 handle_* init 路径）：
i2s_channel_enable(tx_chan);  // 已有
i2s_channel_enable(rx_chan);  // 新增

// 退出子功能时（各 exit 弹窗确认路径）：
audio_running = false;
if (audio_task_handle) { vTaskDelay(pdMS_TO_TICKS(100)); }
i2s_channel_disable(tx_chan);
i2s_channel_disable(rx_chan);  // 新增
```

### 3.3 麦克风采集任务（最小验证代码）

```cpp
static void mic_capture_task(void* arg) {
    int16_t samples[256];
    size_t bytes_read;
    while (true) {
        i2s_channel_read(rx_chan, samples, sizeof(samples), &bytes_read, portMAX_DELAY);
        // 简单音量计算（验证数据通路）
        int32_t sum = 0;
        for (int i = 0; i < 128; i++) sum += abs(samples[i]);
        int avg = sum / 128;
        ESP_LOGI(TAG, "MIC level: %d", avg);
    }
}
```

## 5. 注意事项

1. **采样率对齐**：麦克风和扬声器共享 WS，必须相同采样率（22050Hz）
2. **INMP441 L/R**：L/R 引脚接地 = 左声道。MONO 模式下只读左声道数据
3. **24→16 bit 截断**：INMP441 输出 24-bit，I2S 配 16-bit 时硬件自动取高 16 位，语音应用影响可忽略
4. **Wi-Fi 不冲突**：I2S 使用独立外设，不受 ADC2 的 Wi-Fi 共用限制
5. **`portMAX_DELAY`**：当前播放任务用 `portMAX_DELAY` 阻塞写，建议改为 `pdMS_TO_TICKS(100)` 防止与麦克风读死锁
