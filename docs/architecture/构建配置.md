## 2. 关键 sdkconfig 设置

> 可通过 `idf.py menuconfig` 可视化配置，或直接查看 `sdkconfig` 文件。

### 2.1 芯片与 PSRAM

```
CONFIG_IDF_TARGET="esp32s3"
CONFIG_SPIRAM=y                        # 启用 PSRAM
CONFIG_SPIRAM_MODE_OCT=y               # Octal SPI PSRAM (8MB)
CONFIG_SPIRAM_SPEED_80M=y              # PSRAM 时钟 80MHz
CONFIG_SPIRAM_USE_MALLOC=y             # malloc() 自动从 PSRAM 分配
```

> ⚠️ **必须启用 PSRAM**：GIF 28 帧需 ~2.24 MB，远超内部 DRAM (~320 KB)。
>
> 在 `menuconfig` 中路径：`Component config` → `ESP PSRAM`

### 2.2 CPU 频率

```
CONFIG_ESP_DEFAULT_CPU_FREQ_MHZ_240=y  # 240MHz
```

### 2.3 SPIFFS

```
CONFIG_SPIFFS_MAX_PARTITIONS=1
```

### 2.4 FreeRTOS

```
CONFIG_FREERTOS_HZ=1000                # 1kHz tick (1ms 精度)
```

## 3. 分区表

文件：`partitions.csv`

| 分区名 | 类型 | 子类型 | 偏移 | 大小 | 用途 |
|--------|------|--------|------|:---:|------|
| nvs | data | nvs | 0x9000 | 24 KB | 非易失性存储 (WiFi 凭证等) |
| phy_init | data | phy | 0xF000 | 4 KB | PHY 初始化数据 |
| factory | app | factory | 0x10000 | **2 MB** | 固件 |
| storage | data | spiffs | — | **4 MB** | 图片/音频资源 |

> Flash 总容量：16 MB。分区布局使固件和资源独立，OTA 升级固件不影响资源。

## 4. 资源文件管理

### 4.1 SPIFFS 镜像生成

顶层 `CMakeLists.txt` 中：

```cmake
spiffs_create_partition_image(storage resource FLASH_IN_PROJECT)
```

- `storage`：分区名，与 `partitions.csv` 中的 storage 分区对应
- `resource`：源文件夹，编译时将 `resource/` 目录内容打包为 SPIFFS 镜像
- `FLASH_IN_PROJECT`：烧录固件时一同烧录 SPIFFS 镜像

> ⚠️ **2026-06 更新**：资源文件已迁移到 SD 卡。SPIFFS 分区仅保留 NVS 配置。详见 [SD卡开发指南.md](../guides/SD卡开发指南.md)。

### 4.2 资源文件命名规范

| 模式 | 示例 | 用途 | 存储位置 |
|------|------|------|----------|
| `%04d.png` | `0001.png` ~ `0009.png` | 图片浏览器 | SD 卡 `/sdcard/` |
| `gif_%04d.png` | `gif_0001.png` ~ `gif_0028.png` | GIF 动画帧 | SD 卡 `/sdcard/` |
| `<W>x<H>.raw` | `500x150.raw` | 走马灯图片 | SD 卡 `/sdcard/` |
| `music.wav` | `music.wav` | 背景音乐 | SD 卡 `/sdcard/` |
| `rec.wav` | `rec.wav` | 录音文件（自动生成） | SD 卡 `/sdcard/` |

### 4.3 资源转换工具

| 脚本 | 功能 |
|------|------|
| `cherry/convert_jpg_to_raw.py` | JPG → RGB565 raw 格式（用于走马灯） |
| `cherry/convert_mp3_to_wav.py` | MP3 → WAV (22050Hz 16-bit Mono) |

### 4.4 运行时访问

```cpp
// SPIFFS 挂载（仅配置数据）
esp_vfs_spiffs_conf_t conf = {
    .base_path = "/spiffs",
    .partition_label = "storage",
    .max_files = 5,
    .format_if_mount_failed = false
};

// SD 卡挂载（资源 + 录音）
sd_card_init();  // 挂载到 /sdcard/

// 文件访问示例
FILE* f = fopen("/sdcard/0001.png", "rb");  // 资源从 SD 卡读取
```

## 5. 项目构建结构

```cmake
# 顶层 CMakeLists.txt
cmake_minimum_required(VERSION 3.16)
include($ENV{IDF_PATH}/tools/cmake/project.cmake)
project(box-demo)
spiffs_create_partition_image(storage resource FLASH_IN_PROJECT)

# main/CMakeLists.txt
idf_component_register(SRCS "main.cpp"
                    INCLUDE_DIRS "."
                    REQUIRES
                        LovyanGFX
                        buttons
                        display
                        storage
                        audio
                        net_test
                        record
                        asr
                        chat
                    )
```

### 依赖关系

```
box-demo
├── LovyanGFX — 图形库 (本地组件)
├── buttons   — 按键驱动
├── display   — TFT + UI 绘制 (依赖 LovyanGFX + storage)
├── storage   — SPIFFS + SD 卡 (依赖 spiffs + driver + sdmmc + vfs + fatfs + esp_timer + esp_driver_sdspi)
├── audio     — I2S0 TX 播放
├── net_test  — WiFi + Ping/HTTP/TCP
├── record    — I2S1 RX 录音 (依赖 audio)
├── asr       — 语音识别上传
├── chat      — AI 对话 + TTS
└── ESP-IDF 内置: spiffs, driver, sdmmc, vfs, fatfs, esp_timer, esp_driver_sdspi, esp_http_client
```

> `LovyanGFX` 存放在 `components/LovyanGFX/`，作为本地组件，不依赖网络下载。

## 6. 编译与烧录

```powershell
# 1. 激活 ESP-IDF 环境
. "D:\EE\ESP-IDF\.espressif\v6.0.1\esp-idf\export.ps1"

# 2. 设置目标芯片 (仅首次)
idf.py set-target esp32s3

# 3. 配置项目 (可选，调整 PSRAM/分区等)
idf.py menuconfig

# 4. 编译
idf.py build

# 5. 烧录 (含 SPIFFS 资源)
idf.py -p COM7 flash

# 6. 查看串口日志
idf.py -p COM7 monitor
```

### 一键编译烧录

```powershell
idf.py build flash -p COM7
```

### 仅烧录 SPIFFS 资源（不改固件时）

```powershell
idf.py storage-flash -p COM7
```

## 7. 调试

### 串口日志

- 波特率：115200 (ESP-IDF 默认)
- 日志级别：INFO (可在 `menuconfig` → `Component config` → `Log output` 中调整)
- 关键日志标签：`box-demo`

### 常见编译问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `SPIFFS mount failed` | 分区表未烧录或资源文件夹为空 | 确认 `resource/` 非空，重新 `idf.py flash` |
| `heap_caps_malloc failed` | PSRAM 未启用 | `menuconfig` → 启用 `ESP PSRAM` |
| `LovyanGFX not found` | 组件路径问题 | 确认 `components/LovyanGFX/` 存在 |
| `drawPng failed` | PNG 文件损坏或格式不兼容 | 确认 PNG 为 RGB/RGBA 8-bit |
