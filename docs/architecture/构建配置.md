## 2. 关键 sdkconfig 设置

> 可通过 `idf.py menuconfig` 可视化配置，或直接查看 `sdkconfig` 文件。

### 2.1 芯片与 PSRAM

```
CONFIG_IDF_TARGET="esp32s3"
CONFIG_SPIRAM=y                        # 启用 PSRAM
CONFIG_SPIRAM_MODE_OCT=y               # Octal SPI PSRAM (8MB)
CONFIG_SPIRAM_SPEED_80M=y              # PSRAM 时钟 80MHz
CONFIG_SPIRAM_USE_MALLOC=y             # malloc() 自动从 PSRAM 分配
```

> ⚠️ **必须启用 PSRAM**：GIF 28 帧需 ~2.24 MB，远超内部 DRAM (~320 KB)。
>
> 在 `menuconfig` 中路径：`Component config` → `ESP PSRAM`

### 2.2 CPU 频率

```
CONFIG_ESP_DEFAULT_CPU_FREQ_MHZ_240=y  # 240MHz
```

### 2.3 存储（SD 卡 + SPIFFS）

```
# SD 卡为主存储（资源 + 录音）
CONFIG_SPIFFS_MAX_PARTITIONS=1    # SPIFFS 仅用于 NVS 配置分区
```

> ⚠️ **2026-06 更新**：资源文件已迁至 SD 卡（`/sdcard/`）。SPIFFS 分区仅保留 NVS 配置用途。详见 [SD卡开发指南.md](../guides/SD卡开发指南.md)。

### 2.4 FreeRTOS

```
CONFIG_FREERTOS_HZ=1000                # 1kHz tick (1ms 精度)
```

## 3. 分区表

文件：`partitions.csv`

| 分区名 | 类型 | 子类型 | 偏移 | 大小 | 用途 |
|--------|------|--------|------|:---:|------|
| nvs | data | nvs | 0x9000 | 24 KB | 非易失性存储 (WiFi 凭证等) |
| phy_init | data | phy | 0xF000 | 4 KB | PHY 初始化数据 |
| factory | app | factory | 0x10000 | **2 MB** | 固件 |
| storage | data | spiffs | — | **4 MB** | SPIFFS（NVS 配置，非资源） |

> Flash 总容量：16 MB。资源文件（图片/音频/录音）存储在 SD 卡，不占用 Flash 分区。

## 4. 资源文件管理

资源文件存储在 SD 卡（FAT32），编译时不需要打包进固件。

### 4.1 SD 卡目录结构

```
/sdcard/
├── img/           # 图片浏览器 (0001.png ~ 0009.png)
├── gif/           # GIF 动画帧 (frame_00.png ~ frame_27.png)
├── 500x150.raw    # 走马灯图片
├── music.wav      # 背景音乐 (22050Hz Mono 16-bit)
├── rec.wav        # 录音文件（运行时生成）
└── chat_reply.wav # Chat TTS 音频（运行时生成）
```

### 4.2 资源文件命名规范

| 模式 | 示例 | 用途 | 存储位置 |
|------|------|------|----------|
| `%04d.png` | `0001.png` ~ `0009.png` | 图片浏览器 | `/sdcard/img/` |
| `frame_%02d.png` | `frame_00.png` ~ `frame_27.png` | GIF 动画帧 | `/sdcard/gif/` |
| `<W>x<H>.raw` | `500x150.raw` | 走马灯图片 | `/sdcard/` |
| `music.wav` | `music.wav` | 背景音乐 | `/sdcard/` |
| `rec.wav` | `rec.wav` | 录音文件（运行时生成） | `/sdcard/` |

### 4.3 资源转换工具

| 脚本 | 功能 |
|------|------|
| `cherry/convert_jpg_to_raw.py` | JPG → RGB565 raw 格式（用于走马灯） |
| `cherry/convert_mp3_to_wav.py` | MP3 → WAV (22050Hz 16-bit Mono) |

### 4.4 运行时访问

```cpp
// SD 卡挂载（资源 + 录音）
sd_card_init();  // 挂载到 /sdcard/

// 文件访问示例
FILE* f = fopen("/sdcard/img/0001.png", "rb");  // 资源从 SD 卡读取
```

## 5. 项目构建结构

```cmake
# 顶层 CMakeLists.txt
cmake_minimum_required(VERSION 3.16)
include($ENV{IDF_PATH}/tools/cmake/project.cmake)
project(box-demo)
spiffs_create_partition_image(storage resource FLASH_IN_PROJECT)

# main/CMakeLists.txt
idf_component_register(SRCS "main.cpp"
                    INCLUDE_DIRS "."
                    REQUIRES
                        LovyanGFX
                        buttons
                        display
                        storage
                        audio
                        net_test
                        record
                        asr
                        chat
                    )
```

### 依赖关系

```
box-demo
├── LovyanGFX — 图形库 (本地组件)
├── buttons   — 按键驱动
├── display   — TFT + UI 绘制 (依赖 LovyanGFX + storage)
├── storage   — SPIFFS + SD 卡 (依赖 spiffs + driver + sdmmc + vfs + fatfs + esp_timer + esp_driver_sdspi)
├── audio     — I2S0 TX 播放
├── net_test  — WiFi + Ping/HTTP/TCP
├── record    — I2S1 RX 录音 (依赖 audio)
├── asr       — 语音识别上传
├── chat      — AI 对话 + TTS
└── ESP-IDF 内置: spiffs, driver, sdmmc, vfs, fatfs, esp_timer, esp_driver_sdspi, esp_http_client
```

> `LovyanGFX` 存放在 `components/LovyanGFX/`，作为本地组件，不依赖网络下载。

## 6. 编译与烧录

```powershell
# 1. 激活 ESP-IDF 环境
. "D:\EE\ESP-IDF\.espressif\v6.0.1\esp-idf\export.ps1"

# 2. 设置目标芯片 (仅首次)
idf.py set-target esp32s3

# 3. 配置项目 (可选，调整 PSRAM/分区等)
idf.py menuconfig

# 4. 编译
idf.py build

# 5. 烧录
idf.py -p COM7 flash

# 6. 查看串口日志
idf.py -p COM7 monitor
```

### 一键编译烧录

```powershell
idf.py build flash -p COM7
```

## 7. 调试

### 串口日志

- 波特率：115200 (ESP-IDF 默认)
- 日志级别：INFO (可在 `menuconfig` → `Component config` → `Log output` 中调整)
- 关键日志标签：`box-demo`

### 常见编译问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `SD card mount failed` | SD 卡未插入或格式不对 | 确认 FAT32 格式，重新插入 |
| `heap_caps_malloc failed` | PSRAM 未启用 | `menuconfig` → 启用 `ESP PSRAM` |
| `LovyanGFX not found` | 组件路径问题 | 确认 `components/LovyanGFX/` 存在 |
| `drawPng failed` | PNG 文件损坏或格式不兼容 | 确认 PNG 为 RGB/RGBA 8-bit |
